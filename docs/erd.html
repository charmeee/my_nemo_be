마이그레이션 히스토리

V1
초기 스키마 — users / albums / album_members / images / notifications / trash / invite_links + TLDraw 테이블 (tldraw_documents / album_pages)
V2
enum CHECK 제약 대문자화 — JPA EnumType.STRING 호환 (ADMIN/EDITOR/VIEWER 등)
V3
jsonb → text 전환 — Hibernate 7.x String 매핑 호환 (tldraw_documents.state / notifications.payload / trash.original_data)
V4
users.password_hash 추가 — 이메일 / 비밀번호 로그인 지원
V5
excalidraw_pages 추가 — TLDraw → Excalidraw 마이그레이션 (legacy 테이블은 호환성을 위해 유지)
V6
images.excalidraw_file_id 추가 — Excalidraw fileId(SHA-1) ↔ url 역매핑 인덱스 (album_id, excalidraw_file_id)
